# 知人（zhiren）· 干部信息管理系统

面向组织人事场景的干部信息管理系统：由**干部组集中维护**，多科室在**局域网**内同步查看、搜索、导出，并支持到龄提醒、批量维护、数据分析等。

- 一次安装在单位内一台 Windows 电脑上（当"服务器"），其他电脑和平板**不用装任何东西**，用浏览器就能访问。
- 所有人看到的是同一份数据，干部组改了，其他科室刷新就能看到。
- 数据全部保存在这台电脑本地，不上网、不传云，适合内网使用。

---

# 一、先搞清楚两个角色

| 叫法 | 是什么 | 怎么做 |
|---|---|---|
| **服务器电脑** | 单位里**固定一台**、平时开着的 Windows 电脑（台式机即可） | 只在**这一台**上安装本系统 |
| **使用电脑 / 平板** | 各科室自己的 Windows 电脑、华为平板等 | **什么都不用装**，打开浏览器输入网址即可 |

> 简单说：**装一台，大家用浏览器连过来。** 关键就是知道"服务器电脑"的网址（IP 地址）。

---

# 二、在"服务器电脑"上安装（只做一次）

1. 把安装包 `zhiren-setup-vX.X.X.exe` 拷到这台电脑上。
2. **双击**安装包，一路点"下一步"，完成。（装在默认位置即可）
3. 安装完成后，桌面会出现一个**"知人"**图标。
4. 双击桌面"知人"图标启动。会弹出一个**黑色小窗口**（这是后台服务，**别关它**，关了系统就停了），并自动打开浏览器进入系统。
5. **首次启动时，Windows 可能弹出"是否允许此程序访问网络"** —— 一定要点 **"允许访问"**（公用和专用网络都勾上）。不允许的话，其他电脑就连不进来。

> 想让它关机重启后自动运行？把桌面"知人"图标**复制**一份，放进"启动"文件夹（开始菜单 → 所有程序 → 启动）。或让单位 IT 帮忙设置。

---

# 三、查"服务器电脑"的网址（IP 地址）——重点

其他电脑/平板要访问，必须知道服务器电脑的 IP。两种查法，任选其一：

### 方法 A：看启动时的黑色小窗口（最简单）
启动"知人"后，那个黑色小窗口里会有这样几行字：

```
知人 · 干部信息管理系统 已启动：http://localhost:8080
局域网其他设备可访问：http://192.168.1.248:8080
```

**`局域网其他设备可访问` 后面那一串，就是要告诉大家的网址。** 例如上面就是 `http://192.168.1.248:8080`（你那台的数字会不一样）。

### 方法 B：用 ipconfig 命令查
1. 按键盘 **Win 键 + R**，输入 `cmd`，回车，打开黑色命令窗口。
2. 输入 `ipconfig`，回车。
3. 找到 **"IPv4 地址"** 那一行，后面形如 `192.168.1.248` 的就是本机 IP。
4. 访问网址 = `http://` + 这个 IP + `:8080`，即 `http://192.168.1.248:8080`。

> ⚠️ **IP 可能会变**：如果这台电脑重启后 IP 变了，网址也要跟着变。为避免麻烦，建议请单位 IT 给服务器电脑设一个**固定 IP**（静态地址），以后网址就永远不变。

---

# 四、其他电脑 / 平板怎么访问

前提：使用的电脑/平板和"服务器电脑"在**同一个局域网**（连同一个路由器/交换机，平板连单位同一个 Wi-Fi）。

### Windows 电脑
1. 打开 **Chrome、Edge 或 360 安全浏览器（极速模式）**。（⚠️ 不要用 IE，IE 打不开）
2. 在地址栏输入服务器网址，例如 `http://192.168.1.248:8080`，回车。
3. 出现登录界面就成功了。可以把这个网址**存为收藏夹**，下次直接点。

### 华为平板
1. 打开平板自带的**华为浏览器**（或 Chrome）。
2. 确认平板连的是**单位的 Wi-Fi**（和服务器电脑同一个网）。
3. 地址栏输入同样的网址 `http://192.168.1.248:8080`，回车即可。

# 五、首次登录与建账号

- 默认管理员账号：**用户名 `admin`，密码 `admin123`**。
- 登录后请**立即点右上角"改密"修改密码**。
- 管理员进入 **"系统管理 → 账号管理"** 给各科室建账号，并分配角色：

| 角色 | 能做什么 |
|---|---|
| **管理员** | 管账号、改退休规则、改设置、备份恢复 |
| **维护者**（干部组） | 新建/编辑/删除干部、批量维护、导入 —— 以及查看者的全部 |
| **查看者**（其他科室） | 只能搜索、查看、导出，不能修改 |

# 六、日常使用

- **查询**：顶部"查询"。上面是全局快查框（输入姓名/职务/简历词条），下面"高级组合筛选"可多条件叠加。搜索是**精准匹配**——"张锋"不会搜出"张峰"。
- **新建/编辑干部**（维护者）：填写约 40 项信息，可添加多个家庭成员；年龄、工作年限、到龄时间由系统**自动算出**，不用手填。
- **到龄提醒**：登录首页会弹出本期到龄人数；"到龄提醒"页可看名单、**导出名单**（Excel/CSV）。
- **批量维护**（维护者）：先筛人、勾选，再统一设置某字段（如把一批人的"年度考核情况"设为"优秀"）。
- **导入/导出**：导出得到 Excel 可打开的表格；导入支持把 Excel 另存的 CSV 文件批量导入（建议先导出一份当模板照着填）。
- **数据分析**：按性别、年龄结构、学历、编制、未来五年到龄等出统计。

# 七、数据安全与备份

- 所有数据保存在**服务器电脑本地**：`C:\ProgramData\zhiren\data\zhiren.json`。
- 系统**每天自动备份**（默认保留 30 份），备份在同目录的 `backups` 文件夹里。
- 管理员可在 **"系统管理 → 备份与恢复"** 里"立即备份"或"一键恢复到某个备份"。
- **建议**：定期把 `C:\ProgramData\zhiren` 整个文件夹拷到 U 盘/移动硬盘，多一重保险。

# 八、常见问题

**Q：其他电脑打不开 / 转圈连不上？**
1. 确认服务器电脑上那个黑色小窗口还开着（没关）。
2. 确认两台在同一局域网，网址里的 IP 没输错、端口是 `:8080`。
3. 多半是**防火墙**拦了：在服务器电脑上重新启动一次"知人"，看到弹窗时点"允许访问"；或让 IT 在防火墙放行 8080 端口。
4. 用浏览器先在**服务器电脑本机**打开 `http://localhost:8080`，能开说明系统正常，问题在网络/防火墙。

**Q：页面打不开、显示要换浏览器？** 说明用了 IE。请改用 Chrome / Edge / 360 极速模式。

**Q：网址过几天连不上了？** 可能服务器电脑 IP 变了，重新用"第三节"的方法查一次新 IP。建议设固定 IP 一劳永逸。

**Q：服务器电脑关机/重启了会怎样？** 系统会停，数据不会丢。重新双击桌面"知人"图标启动即可。

**Q：忘了管理员密码？** 联系系统维护方处理（涉及直接操作数据文件）。

Go 单文件 exe（内嵌 Web 与前端，纯标准库）+ JSON 文件存储（原子写入）+ 服务端渲染的轻量网页（兼容老 Chromium）。详见 ADR-0004 / 0005。

## 本地运行（需 Go 1.20）
```bash
go run .                 # 默认 :8080，数据写到 ./data/zhiren.json
go run . -addr :9000     # 自定义端口
```

推送 `v*` 标签触发 GitHub Actions：Windows runner 上用 Go 1.20 编译 exe → Inno Setup 打包 → 发布到 Releases。
```bash
git tag v1.0.0 && git push origin v1.0.0
```

## 关于"绿色浏览器随包打入"
安装脚本已预留接口：构建时若 `dist/browser/chrome.exe` 存在，会被一并打包，且桌面快捷方式自动用它打开（`-browser` 参数）。需选定一个 **兼容 Win7 的便携 Chromium（约 Chrome 109 一档）** 放入即可启用。
