# 退休年龄做成可配置规则，不硬编码

"到龄时间"既是干部档案的一个字段，又是"到龄提醒"功能的触发依据，而退休年龄并非固定数字：它随性别、干部/工人身份（经典规则男 60、女干部 55、女工人 50）、可能还随级别变化；且自 2025 年起国家实行渐进式延迟退休，法定退休年龄改为按出生年月逐月推迟的动态对照表。因此我们决定：由管理员维护一张退休年龄规则表（性别 × 身份 × 可选级别 → 年龄），内置渐进式延迟退休对照表（按出生年月套用），并允许对个别特殊干部手工覆盖到龄时间。政策再变只改配置、不改程序。

## Consequences

- 到龄时间默认由系统按规则算出，但保留手工覆盖入口处理特例。
- 需要一个管理退休年龄规则表与延迟退休对照表的配置界面。
