# 部署目标为物理隔离内网，安装包完全离线自包含，GitHub 仅作维护者分发中转

干部信息含身份证号、家庭成员、政治面貌等敏感数据，部署现场是物理隔离内网，无法访问外网。因此安装包必须完全离线、自包含：不依赖联网下载运行时、在线字体或 CDN，不连任何云服务，且不支持在线自动更新。GitHub（gh）在本项目中**仅**作为维护者（你）获取发布产物的渠道——从 gh Releases 下载安装包后，经移动介质带入内网交付，部署现场连不上 gh。

## Consequences

- 升级靠重新分发离线安装包 + 手工替换，没有在线热更新。
- 任何第三方依赖（浏览器、运行时、字体）都必须随包打入，不能运行时拉取。
